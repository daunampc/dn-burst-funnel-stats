== Requirements ==
* Burst Statistics
* WooCommerce

== GitHub updates ==
This plugin includes a GitHub release updater.

1. Create a GitHub repository, for example: toshstack-dev/dn-burst-funnel-stats
2. In dn-burst-funnel-stats.php, update DN_BURST_FUNNEL_STATS_GITHUB_REPO if your repository is different.
3. Increase the plugin Version header and DN_BURST_FUNNEL_STATS_VERSION constant.
4. Create a GitHub release with tag v1.0.1, v1.0.2, etc.
5. Attach the plugin ZIP to the release for best results.
